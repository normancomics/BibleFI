
# Sound Files for Bible.Fi

This directory contains all sound effect files used in the Bible.Fi application. We're using free sound effects with 8-bit/retro arcade style.

## Sound Files:
- coin.mp3: Coin collection sound (used for tithes/donations)
- click.mp3: UI button click sound
- error.mp3: Error notification sound
- scroll.mp3: Scroll/paper unfurling sound
- select.mp3: Selection sound
- powerup.mp3: Power-up sound (for achievements)
- success.mp3: Success notification sound

## Sources:
These sounds were obtained from free sound libraries. For production use, ensure proper licensing.
